What is IoT?

IoT stands for Internet for things. We can use it to communicate devices using internet.

Which module is used?

The ESP32 DEVKIT V1 Board is used. it has 38 GPIO(General Purpose Input/Output) pins and inbuilt IoT chip.

How it works?

The ESP gets data from Blynk. Blynk is a library which provides IP address and helps to host on the internet. A Virtual pin on blynk app turns on or off the relay(A switch controlled by microcontrollers).

The Connections:

The + pin of relay is connected on 5V(positive power) pin of ESP. The - pin of relay is connected on gnd(negative power) pin of ESP. The Data pin of relay is connected on GPIO2(GPIO Pin) pin of ESP.
